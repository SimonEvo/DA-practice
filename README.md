# 三周数据分析学习计划（电商方向 · FineBI版本）

## 总目标
在 3 周内完成一份可直接用于社招的数据分析作品集，包括：
- SQL 销售指标体系项目
- FineBI 多页商业报表（含交互大屏）
- 商业洞察分析
- GitHub 项目仓库
- 数据分析师简历强化

# 第 1 周：SQL（核心能力周）

**目标：** 熟练工作中 80% 的 SQL 任务，并完成一个项目  
**输出物：** 《SQL 构建电商销售指标体系项目》

---

## Day 1：理解电商数据结构（30 min）
掌握企业中常见的四表结构：

- user_table（用户）
- item_table（商品）
- order_table（订单主表）
- order_detail_table（订单明细）

---

## Day 2：生成 10 万条模拟数据（30–40 min）
- 使用 Python 生成电商模拟数据
- 熟悉字段含义、外键关系

---

## Day 3：SQL · JOIN 练习
- INNER JOIN / LEFT JOIN / RIGHT JOIN
- 订单 + 用户
- 订单 + 商品

---

## Day 4：SQL · 聚合分析
常见业务指标：

- GMV（交易额）
- 订单量
- 售卖商品数
- 客单价（AOV）
- 品类销售额排名

---

## Day 5：SQL · 窗口函数
- 品类 TOP10 排名（RANK/DENSE_RANK）
- 用户最近下单日期
- 用户生命周期价值 LTV
- 留存率分析

---

## Day 6：SQL · CTE + CASE WHEN
构建指标体系：

- 新客 / 老客
- 复购率
- 用户分层
- 漏斗指标（浏览→下单→支付）

---

## Day 7：产出 Week1 项目
内容包括：

- 数据结构图
- SQL 查询代码
- 指标解释
- 查询结果截图

# 第 2 周：FineBI（核心作品集周）

**目标：** 使用 Week1 的数据制作完整的 BI 报表  
**输出物：** 《电商商业分析大屏（FineBI）》  
（大屏 + 多页图表视图）

---

## Page 1：总览（Overview）
KPI 指标卡：

- GMV
- 订单数
- 客单价
- 售卖商品数

图表：

- GMV 日/周/月趋势图
- GMV 环比/同比柱状图
- 订单增长趋势

---

## Page 2：商品分析（Product Analytics）
- 品类销售额占比（饼/矩形树图）
- 品牌渗透率
- TOP10 商品销量榜
- 商品销售趋势折线图
- 价格带销量分布（直方图）

---

## Page 3：用户分析（User Analytics）
- 用户画像（性别/城市/年龄）
- 新客 vs 老客 GMV 对比
- 留存率折线图
- 复购率趋势
- 用户消费等级分布（雷达图）

---

## Page 4：运营分析（Operations）
- 各城市销售额地图
- 各渠道 GMV 对比
- 时段销售热力图
- 退货率分析

---

## Page 5：BI 大屏（FineBI 特有）
